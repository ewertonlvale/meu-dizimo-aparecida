3. Descrição do Serviço

3.1 Funcionalidades

O Meu Dízimo Aparecida oferece:

  • Orientação sobre Devolução: Informações sobre como devolver o dízimo
  • Geração de QR Code PIX: Código para pagamento via PIX
  • Registro de Comprovantes: Envio e armazenamento de comprovantes
  • Atendimento Pastoral: Encaminhamento para secretaria quando necessário

3.2 Disponibilidade

O serviço está disponível:

4.1 Requisitos de Cadastro

Para se cadastrar como dizimista, você deve:

4.2 Responsabilidade pelos Dados

Você é responsável por:

4.3 Unicidade do Cadastro

Cada número de WhatsApp pode ter apenas um cadastro ativo. Caso identifiquemos duplicidade:

  • Podemos suspender temporariamente o acesso

5. Devolução do Dízimo

5.1 Natureza da Contribuição

É importante esclarecer que:

  • O dízimo é uma contribuição voluntária e religiosa
  • Não constitui pagamento por serviços prestados
  • É uma prática de fé e não uma obrigação legal
  • O valor é determinado livremente pelo dizimista

5.2 Processo de Devolução

Ao devolver o dízimo através do Meu Dízimo Aparecida:

  • Você receberá um QR Code PIX padrão da comunidade
  • O pagamento é processado diretamente pela instituição bancária
  • O Meu Dízimo Aparecida NÃO processa pagamentos
  • Após o pagamento, você pode enviar o comprovante voluntariamente

5.3 Comprovantes

  • O envio de comprovante é opcional mas recomendado
  • Comprovantes são verificados manualmente pela secretaria
  • Não validamos automaticamente comprovantes
  • Comprovantes podem ser solicitados para fins de controle interno

5.4 Não Processamento Automático

⚠️ Atenção: O Meu Dízimo Aparecida é apenas um assistente de orientação. NÃO processamos pagamentos automaticamente, NÃO temos acesso a dados bancários e NÃO validamos transações automaticamente.
